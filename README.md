# 🍃 LeafNodeCalculator 🎵

一个基于初音未来主题的叶子节点计算器，适用于手机浏览。

## 🌸 功能

- 允许用户自定义树的度数。
- 简洁美观的初音未来风格设计。
- 响应式布局，适合手机和平板电脑浏览。
